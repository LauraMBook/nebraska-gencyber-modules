# Lesson Plan Introduction to First Principles

### Summary


### Grade
High School

### Time Required
30 Minutes

### First principles
- [x] Domain Separation
- [x] Process Isolation
- [x] Resource Encapsulation
- [x] Modularity
- [x] Least Privilege
- [x] Abstraction
- [x] Data Hiding
- [x] Layering
- [x] Simplicity
- [x] Minimization

1. Students will be able to identify Cybersecurity Principles
